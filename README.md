# GujPsych Repository

GujPsych Repository is an academic initiative focused on developing and organizing psychiatric and psychological resources in Gujarati. The platform aims to facilitate access to validated psychometric tools, educational material, and clinically relevant resources for students, clinicians, and researchers.

## Purpose

* To provide Gujarati adaptations of psychometric scales
* To support academic and clinical use in psychiatry
* To promote culturally relevant mental health resources

## Current Status

This website is under active development update  per availability and submission of resources. Content and features are being continuously updated.
